# Go-Snake

Snake made in Go with raylib (for fun).

## Key Bindings

| Key | Function     |
|-----|--------------|
| w   | Move Up      |
| a   | Move Left    |
| s   | Move Down    |
| d   | Move Right   |
| r   | Restart Game |
| p   | Toggle Pause |
| v   | Toggle VSync |
| f   | Toggle FPS   |

## Build

Run `make` to build.
